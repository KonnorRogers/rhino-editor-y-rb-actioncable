# Prerequisites

- Docker
- Ruby
- Node
- PNPM


## Installation


```bash
git clone https://github.com/konnorrogers/rhino-editor-y-rb-actioncable.git
cd actioncable-yrb-collaboration
bundle install
pnpm install
```

## Running the server

```bash
overmind start -f Procfile.dev
```

Navigate to: `localhost:5100`
